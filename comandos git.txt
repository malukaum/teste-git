1 - Criar ou logar na conta do github
2 - Instalar Github -> https://git-scm.com/download/win
3 - Instalar Github Desktop
4 - Criar um repositorio remoto -> https://github.com/
5 - Iniciar o repositorio localmente -> git init
6 - Criar um arquivo
7 - Adicionar ao stagging -> git add <nome do arquivo>
8 - Configurar nome de usuario -> git config --global user.name <nome de usuario>
9 - Configurar email da conta -> git config --global user.email <email>
10 - Configurar repositorio do git -> git remote add origin <caminho do repositorio>
11 - Commitar o arquivo -> git commit -m <mensagem do commit>
12 - Subir o arquivo -> git push origin master
13 - Criar uma branch nova -> git branch <nome da branch nova>
14 - Mudar para branch nova -> git checkout <nome da branch nova>
15 - Commitar novas atualizacoes -> git commit -m <mensagem do commit>
16 - Subir novas atualizacoes -> git push origin <nome da nova branch>
